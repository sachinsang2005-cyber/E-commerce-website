A full-stack **Farsan / Namkeen E-commerce Web Application** built using **Node.js, Express, MongoDB, and EJS**.

This project allows users to browse, add, edit, and manage traditional farsan food products with image upload support.

---

## Tech Stack

### Frontend
- EJS (Server-side rendering)
- HTML, CSS
- Bootstrap

### Backend
- Node.js
- Express.js
- MongoDB (Mongoose)
- Passport.js (Authentication)
- Express Session
- Connect Flash
- Multer (Image Upload)
- Method-Override

---

## Features

### User Features
- View all farsan products
- View product details
- User signup & login
- Leave product reviews

### Admin / Owner Features
- Add new products
- Upload product images
- Edit existing products
- Delete products
- Authorization (only owner can edit/delete)

---

## Core Functionalities

- CRUD Operations (Create, Read, Update, Delete)
- Authentication & Authorization
- Session Management
- Flash Messages (Success / Error)
- Image Upload using Multer
- RESTful Routing
- Centralized Error Handling
